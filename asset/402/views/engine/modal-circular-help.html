Reading circulars


ads Flyer view

On this view, mouse over or tap an item to display interactive buttons. Click the pagination buttons to change page. With a touch-enabled device, swipe left/right on image to switch between pages.

Non-interactive flyer will denote as: * image only/non-interactive flyer


list List view

On this view, use the search textbox to quickly filter for sale specials. Click or tap on the item text description to see more information. Sort and filter by Department, Brand, and more...


text Text-only view

This friendly view allow for faster loading of sale items. It has all feature of list view but with no image.
